Making corrections

Authorised users can help improve the dataset by correcting the transcribed text.

Authorisation

Authorisation is via GitHub. To become an authorised user, you have to be added as a member of the ANU Stock Exchange Project:

  • Create a GitHub account (if you don’t have one)
  • Contact ??? with your GitHub details and ask to be added to the project
  • Go to the database and click on the ‘Sign in with GitHub’ link at the top right.
  • Follow the prompts to authorise your account.

Editing a row

Splitting a row

Sometimes multiple rows are transcribed as a single block of text. To split the text into individual rows, you can add new rows and divide the content between them. New rows are added as 'sub rows' connected to their parent row. To split a row:

  • Go to the row you want to split by clicking on the link in the search results
  • Click on the View full column button
  • If you're logged in, you should see buttons at the end of each row
  • Click on the button at the end of the row you want to split
  • The add row form displays the contents of the parent row and any existing sub rows. Enter the contents of the new sub row in the text boxes and click Add row

Repeat this process until the contents of the parent row has been split between the sub rows. At this point you can edit the parent row to remove the duplicate content.
